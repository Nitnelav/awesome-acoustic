Hello! I'm Valentin, an enthusiast of open-source and acoustic research. I created Awesome Acoustic to help the community discover, share, and contribute to the best open-source acoustic software and resources. My goal is to make it easier for researchers, engineers, and hobbyists to find and collaborate on high-quality acoustic tools.

Want to add a new open-source acoustic project to this list? Follow these steps:

View this project on GitHub

  1. Fork this repository on GitHub.
  2. Edit the projects.json file and add your project in the appropriate category. Please follow the existing format for consistency.
  3. Commit your changes and open a Pull Request (PR) with a short description of your project.
  4. Once your PR is reviewed and merged, your project will appear on the website and in the README!

If you need help, check the existing entries for examples or open an issue for guidance.

If you have suggestions or want to connect, feel free to open an issue or pull request on the GitHub repository!
